Managing API Builder Applications

To view the details of an API Builder application, select it from the Apps menu. Or, select All Projects to view a list of all applications from which to select.

View API Builder Application Information and Analytics

For API Builder applications, the Overview tab displays creation, publication, and build status information as well as the allocated container points and the server count. The App Server Settings configuration can be updated by clicking the Pencil icon.

Change App Service Settings

To update the App Service Settings:

  1. Click the Pencil icon.

  2. Update the Container Type by selecting a container type from the drop-down menu. The selectable container types are:

    • Dev

    • Small

    • Medium

    • Large

    • XLarge

  3. Update the Min Container Count.

  4. Update the Max Container Count.

  5. Enter any Environmental Variables.

  6. Click the Update Config button to update your App Service Settings and return to the Overview tab. Click the Cancel button to return to the Overview tab without updating your App Service Settings.

View Analytics

The Analytics tab displays the number of API Builder requests over the selected time period as the number of requests per path.

View Connected Applications

The Apps tab displays a list of applications (Titanium or native) that are consuming the selected API Builder app and Mobile Backend Services. Select an application from the list to manage it.

View Publish History

The Publish History tab displays the publication history of your application.

View and Filter API Logs

The Logs tab displays a paginated table of all API access or application calls made by your application to the selected API Builder application service.

The Logs table includes the following fields:

  • URL – Requested URL.

  • Remote IP – IP of the client that made the request.

  • Response Time – Time spent responding to the request.

  • Container – ID of the application server container used to handle the request.

  • Date/Tim – Date and time of the request in UTC.

View Documentation

The Documentation tab displays a list of links to the API Builder guides and to other resources.
