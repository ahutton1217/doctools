Appcelerator CLI 7.0.1.GA - 21 December 2017

Appcelerator CLI 7.0.1.GA is a patch release that includes improvements and bug fixes.

Component versions

The following components are shipped with CLI 7.0.1:

Alloy

1.10.10

API Builder

3.0.0

Cloud CLI

2.0.8

Titanium CLI

5.0.14

Improvements

  • CLI-1289 - Update AppPreview hook

Fixed issues

  • CLI-1290 - npmconf causing errors when resolving plugins
