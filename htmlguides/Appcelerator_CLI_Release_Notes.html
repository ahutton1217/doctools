Appcelerator CLI Release Notes

Appcelerator CLI 7.0.9.RC - 13 December 2018

Appcelerator CLI 7.0.9.RC is a patch release that includes one bug fix.

As of this GA release, the previous CLI patch release is no longer supported. End of support for this version will be up to 2019-06-13 or until the next patch release. Note: major and minor releases continue to be supported according to their nominal lifetime. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.

The following components are shipped with CLI 7.0.9:

Fixed issues

  • ALOY-1638 - Theme is "null" when a theme is defined in config.json
