Alloy Framework

This documentation covers the Alloy framework, a new Appcelerator framework for the Titanium SDK designed to rapidly develop quality Titanium applications. It is based on the model-view-controller architecture and contains built-in support for Backbone.js and Underscore.js.

Review the Alloy Release Notes to see what has changed.
