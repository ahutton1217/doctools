Appcelerator CLI 5.1.0 - 20 November 2015

Appcelerator CLI 5.1.0 is a minor release that includes improvements and bug fixes.

As of this release, CLI 5.0.x will not be supported six months from 5.1.0.GA's release date. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.
