API Builder Connectors

This page is for the beta release of API Builder 1.10.0. Some information may be missing or incomplete.

Connectors are adaptors to allow you to read and write data to and from an external data source, such as Mobile Backend Services, MySQL, Salesforce, and MongoDB, or in server memory. You can install a pre-built connector or create your own.
