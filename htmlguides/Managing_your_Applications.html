Managing your Applications

To manage an application, click the Apps menu top navigation bar and select it from the list of recent apps, search for it by name or GUID, or select All to view a list of all applications in the currently selected organization.

From the Apps menu, you can select either:

  • Client Applications – Titanium, Android or iOS application

  • API Builder Applications

  • Managing Mobile Backend Services Datasources (or legacy ACS application) – only standalone Mobile Backend Services (MBS) datasources (not created with a client application). For MBS datasources created with a client application, access the MBS datasource using the client application's API Builder tab.
