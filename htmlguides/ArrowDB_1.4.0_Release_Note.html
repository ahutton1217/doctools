ArrowDB 1.4.0 - 12 January 2016

This release of ArrowDB includes version 1.1.0 of the CLI and 1.4.0 of the server, and includes a new feature and improvement.

Improvement

  • Size and exclusive containers: Arrow Cloud supports deploying app to an exclusive base container and setting the size of the container in this case.
