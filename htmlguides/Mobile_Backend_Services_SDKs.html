Mobile Backend Services SDKs

In addition to the REST APIs, Mobile Backend Services provides several SDKs you can use to integrate Mobile Backend Services into your own native applications.

  • Titanium applications can easily integrate with Mobile Backend Services using the ti.cloud module, which ships with the Titanium SDK.

  • Android applications can use the Mobile Backend Services Android SDK to integrate with Mobile Backend Services.

  • iOS applications can the use Mobile Backend Services iOS SDK to integrate with Mobile Backend Services.

  • Node.js applications can the use Mobile Backend Services Node SDK to integrate with Mobile Backend Services.
