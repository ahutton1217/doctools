Studio 4.4.0 Beta - 28 October 2015

Studio 4.4.0 is a minor release that includes new features, improvements, and bug fixes. See the full list of issues address in Release 4.4.0.

As of this release, Studio 4.3.x will not be supported six months from 4.4.0.GA's release date. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.

New Features

  • Add option for Studio to remember your username

  • Update Studio to check for minimum Node.js version 0.12.7 and will install version 4.1.0 on clean installs
