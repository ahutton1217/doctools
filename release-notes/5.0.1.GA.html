About This Release

Titanium SDK 5.0.1 is a patch release of the SDK, addressing high-priority issues from previous releases.

Fixed Issues

  • TIMOB-19563 Unable to find Info.plist in root of specified app path
  • TIMOB-19293 Android: Crittercism: UnsatisfiedLinkError on libcrittercism-ndk.so
  • TIMOB-19401 Windows: Missing events for Ti.UI.View
  • TIMOB-19502 Windows: wrong behavior for touch events
  • TIMOB-19533 Forward and reverse geo errors using APIs
  • TIMOB-19541 CLI: Display actual version of installed SDKs with "ti info -t ios" and "ti sdk list"
  • TIMOB-19545 Unable to use CommonJS modules on real device with SDK 5.0.0.GA
  • TIMOB-19549 iOS: JS files in Resources get minified if minification is not "skipped"
  • TIMOB-19550 iOS: Many PNG images causes device/dist builds to fail EAGAIN (too many processes)
  • TIMOB-19554 iOS: pngcrush fails build when crushing an invalid png image
  • TIMOB-19568 iOS: Wrong log message when ATS is explicitly disabled
  • TIMOB-19571 iOS: App names with special characters break the build

API Changes

No APIs were modified for this release.
