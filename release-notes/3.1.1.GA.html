About This Release

This is the Initial Release of the Appcelerator Platform, introducing Appcelerator Studio, the enterprise version of Titanium Studio.

For information about getting started with the Appcelerator Platform, see the Quick Start.

Known Issues

Error enabling Cloud service for project

If you receive a message similar to "Error enabling Cloud service for project" in Studio when trying to enable Cloud services for your project, the ACS server may be down or Studio is unable to connect to the ACS server (APPTS-2358). Try to enable Cloud services later.

If you are still receiving this error when trying to enable Cloud services, see the ACS Troubleshooting Guide.

Appcelerator Studio 3.1.1

This is the Initial Release of Appcelerator Studio.

Appcelerator Studio requires Titanium components to develop mobile applications. By default, Appcelerator Studio automatically checks and prompts you to update these components. To manually check for updates, from the menu bar, select Help > Check for Appcelerator Updates.

This version of Studio is tested against and supports the following Titanium components:
