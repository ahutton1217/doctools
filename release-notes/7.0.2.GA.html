Titanium SDK 7.0.2.GA - 9 February 2018

About this release

Titanium SDK 7.0.2.GA is a patch release of the SDK, addressing high-priority issues from previous releases.

Fixed issues

Android

  • TIMOB-23482 - Android: EmailDialog fails to attach files when using Outlook
  • TIMOB-25259 - Android: Runtime error "exposed beyond app through Intent.getData()" on API 24 & higher when using Ti.Media.previewImage()
  • TIMOB-25506 - Android: ImageView sometimes crashes when loading via URL
  • TIMOB-25518 - Android App Crash due to Attempt to invoke virtual method 'void java.net.HttpURLConnection.disconnect()' on a null object reference
  • TIMOB-25540 - Android: onTrimMemory can cause crash
  • TIMOB-25541 - Android: EmailDialog file attachments not working on Android 7 and higher
  • TIMOB-25565 - Android: Ti.Media.showCamera with an overlay fires success callback multiple times on some devices
  • TIMOB-25597 - Android: Multidexed apps crash on startup on Android 4.x
  • TIMOB-25617 - Android: Push notifications not shown in statusbar on Android 8 if app is backgrounded
  • TIMOB-25620 - Android: SOASTA breaks Android 8.0+ WebView events
  • TIMOB-25621 - Android: Native libraries are not included with compiled Titanium modules
  • TIMOB-25656 - Android. TiViewProxy.getOrCreateView returns null on Titanium 7.0.0+
  • TIMOB-25660 - Android: ListView 'canScroll' property cannot be set dynamically
  • TIMOB-25733 - Android: View with border larger than screen may fail to appear as of 7.0.2.RC
  • TIMOB-25734 - Android : TextField Click event is not firing with 7.0.X SDK
  • TIMOB-25735 - Android: Replace Activity.isDestroyed() with Android 4.1 compatible alternative

iOS

  • TIMOB-17572 - iOS: ActivityIndicator does not reset when used in ListView
  • TIMOB-23634 - iOS: Toolbar on TextField doesn't hide smoothly when calling blur
  • TIMOB-25576 - Ensuring usage of macros bundled core utilities and shells when adding to iOS buildPhases
  • TIMOB-25637 - iOS: Window toolbar animates when opening window
  • TIMOB-25640 - iOS: Ti.Media.VideoPlayer "playbackState" constants are undefined
  • TIMOB-25646 - iOS: After Listview Search overlay not appearing
  • TIMOB-25652 - iOS: Ad-hoc build fails when packaging with WatchApp
  • TIMOB-25672 - iOS: Address 10+ compiler warnings and general issues in source project
  • TIMOB-25674 - iOS Build: Folders beginning with a platform name are excluded from app
  • TIMOB-25675 - iOS11: Button in navigation-bar remains in pressed state when opening new window (Apple bug)
  • TIMOB-25677 - iOS: Ti.Map - App crashes when re-assigning annotations with clustering enabled
  • TIMOB-25680 - iOS 11.2: DocumentViewer not displaying PDF files (Apple regression)
  • TIMOB-25727 - iOS: base64decode issue observed with SDK 7.0.0 and above
  • TIMOB-25738 - iOS: Apple watch simulator not launching
  • TIMOB-25752 - iOS: 7.0.2 RC fails to build on device due to Ti.Media errors

Hyperloop

  • TIMOB-23933 - Hyperloop - Android: JS Wrappers not being generated for org.json .JAR
  • TIMOB-24684 - Hyperloop: Android - Cannot iterate over List interface
  • TIMOB-25604 - Hyperloop: Examples project throws error on iOS

API changes

There were no API changes in this release.
