Release notes

Dashboard 1.1.1 Release Notes

This release of Dashboard includes the following notable bug fixes:

  • When scheduling a push notification, the selected time/date is no longer converted to UTC time. [APPTS-4435]

  • The Collections screen in Cloud > Manage Data has been renamed to Photo Collections to match its function. [APPTS-4175]

  • Manage Data forms now persist unsaved data while creating or editing in one form, and you click a 'Create New object' link in an object selector to create a open a new form. Any unsaved data you entered is preserved when you return to the original form. [APPTS-4158]

    For example, in the following a user in the process of creating a new Post object, and needs to create a new Event object to associated with it. Clicking 'Create a new Event' opens a new form to create the Event. After saving the Event (or canceling the operation), the user is returned to the original Post creation form with any unsaved changes persisted.

Appcelerator Test 7040.75 Release Notes

The Appcelerator Test service and module has been updated to version 7040.75. Appcelerator Studio users will need to install the new module and re-enable the Test service in each of your Studio projects (see Updating the Test module).

Updating the Test module

Appcelerator Studio users need to update the installed Test module to take advantage of this feature, and re-enable Appcelerator Test in each of your Appcelerator Studio projects once the update is complete.

To install the new Test module and re-enable Appcelerator Services:

  1. In Studio select Help > Check for Appcelerator Updates (or re-launch Studio and wait for an update notification to appear).

  2. Click Install in the Appcelerator Updates dialog that appears.

  3. In the tiapp.xml file of each project that is using Test, click Enable Services in the Appcelerator Services section.

    The red × next to Test should change to a green checkmark, as for the other services.

Documentation

The following documentation updates accompany this release:

  • Manage Data documentation describes how to create and edit ACS objects using the Manage Data forms in Dashboard.
  • Appcelerator Test documentation contains new content on using Appcelerator Test to record and run tests.
