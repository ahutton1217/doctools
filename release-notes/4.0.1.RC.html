About This Release

This is a Release Candidate version of Titanium SDK 4.0.1 and should not be used for production. Titanium SDK 4.0.1 is a patch release of the SDK, addressing high-priority issues from previous releases.

Fixed Issues

  • TIMOB-18448 : Android vendor dependency version has to be updated
  • TIMOB-18887 : Android: Wrong implementation of TiResponseCache may cause occasional crashes in Lollipop
  • TIMOB-18916 : iOS: Posting a photo to Facebook gives "Unsupported FBRequest parameter type:TiBlob"
  • TIMOB-18934 : Android: Failed to package application if API Level 21 not installed. Process exit value was 1

API Changes

No APIs were modified for this release.
