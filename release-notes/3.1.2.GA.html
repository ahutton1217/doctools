About This Release

This is a General Availability (GA) release of the Appcelerator Platform, including Appcelerator Studio 3.1.2, addressing high-priority issues in previous releases.

These release notes only address the enterprise features associated with the Appcelerator Platform. For updates about the underlying components of the Appcelerator Platform, see the links in the Titanium Components section below.

Installation Directions

Before upgrading Appcelerator Studio, uninstall the LiveView plugin first. LiveView is now shipped with Appcelerator Studio and may conflict with an existing LiveView plugin installation. To uninstall the LiveView plugin:
  1. Open the About Appcelerator Studio dialog.
    For Mac OS X, from the menu bar, select Appcelerator Studio > About Appcelerator Studio.
    For Linux and Windows, from the menu bar, select Help > About Appcelerator Studio.
  2. In the About dialog, click Installation Details.
  3. Select LiveView Feature, click Uninstall..., then Finish.
After the process completes, restart Studio then start the Studio upgrade process. Do not forget to remove or disable the LiveView update site:
  1. Open the Preferences dialog and navigate to Install/Update > Available Software Sites.
  2. Select the LiveView update site and click either Disable or Remove.

Breaking Changes

Running Multiple Platforms with the Code Analyzer

To run the code analyzer for multiple platforms, select the Code Analysis > Multiple... option from the project's right-click context menu. For more information, see the Code Analyzer guide.

Previously, this option was called Run Selected Platforms.

Enabling LiveView with a Toggle Button

As of this Release, to enable LiveView, toggle the LiveView button on and off to enable or disable LiveView for each run. For more information, see the LiveView guide.

Previously, this feature was controlled in the Run Configurations dialog.

New Features

Export Code Analyzer Results

Results from the Code Analyzer can be exported by right-clicking on the results and selecting Export or selecting Code Analysis > Export Results... from the project's right-click context menu. For more information, see the Code Analyzer guide.

Known Issues

LiveView with Alloy Limitation

When using the LiveView tool with multiple launches of an Alloy project, LiveView only properly updates the last platform you run. For an existing launch with LiveView on another platform, relaunch the application for that platform.

Titanium Components

Appcelerator Studio requires Titanium components to develop mobile applications. By default, Appcelerator Studio automatically checks and prompts you to update the components that are GA. To manually check for updates, from the menu bar, select Help > Check for Appcelerator Updates.
