Dashboard 5.1.2 Release Note

Dashboard 5.1.2 - 14 August 2018

Dashboard 5.1.2 is a patch release, which includes a new feature and bug fixes.

New feature

  • Added a download link for the App Builder Studio 64-bit Windows installer.

Fixed issue

  • Fixed an issue with the Mobile Backend Services Environment dropdown menu not updating to display the selected environment.

  • Fixed an issue with pagination of Mobile Backend Services push notification Subscribed Devices list.

  • Fixed an issue where Mobile Backend Services push notification Subscribed Devices may not have been able to be unsubscribed.
