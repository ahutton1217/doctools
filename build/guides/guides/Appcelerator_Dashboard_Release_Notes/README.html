Appcelerator Dashboard Release Notes

Dashboard 4.15.3 - 5 December 2017

Dashboard 4.15.3 is a patch release, which includes the following a behavior change and an improvement.

Behavior Change

  • The Business Event Service tile on AMPLIFY Home page and product description page have been removed from the Platform.

Improvement

  • Users are no longer required to confirm their email address when using the Password Reset form.
