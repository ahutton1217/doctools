Google Cloud Messaging for Android push notifications

ACS now supports Google Cloud Messaging (GCM) for push notifications on Android devices.

For instructions on setting up GCM, see the "Configure Your App For Enabling Push Notifications" subsection under the "Push Notifications" section of the Android SDK guide.

To send push notifications, refer to PushNotifications.
